Manage strings/words from admin panel; Use as string translator at multilanguage sites;

== Description ==
= With this plugin you are able to: = 

*   Change strings on your wordpress site, from admin panel
*   Plugin works over other language plugins and can be used together
*   Plugin works with generated and optimized files
*   No load on DB while using on site

This plugin uses as less scripts and css as possible to avoid traffic load or performance problem, and is as simple as possible.
Some other features will be added in next releases

== Installation ==

= This section describes how to install the plugin and get it working. =
1. Download and Simple String Manager plugin.
2. Upload the simple-string-manager folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the \'Plugins\' menu in WordPress.
4. You will have a new WP String Manager link in your admin menu, navigate through this menu item
5. Easily search and change words over the site. 
6. To detect new string on admin panel, once reload page where strings appear
7. Enjoy
